# 知情同意书系统

这是一个专业的知情同意书网页应用，用于心理咨询服务中的知情同意流程。

## 功能特点

- ✅ **主页面**：左右两个按钮（咨询知情同意、录音录像知情同意）
- ✅ **同意书页面**：显示同意书全文内容（支持滚动查看）
- ✅ **签名功能**：两个手写签名控件（来访签名、咨询师签名）
- ✅ **预览功能**：双方签名后显示预览版本
- ✅ **导出功能**：保存为图片或PDF
- ✅ **响应式设计**：适配手机、平板、电脑等不同设备
- ✅ **现代、专业的界面**：使用渐变色彩、阴影效果和流畅动画
- ✅ **清晰的代码结构**：分离的HTML、CSS和JavaScript文件

## 项目结构

```
├── index.html          # 主页面和同意书页面结构
├── styles.css          # 现代、专业的界面样式
├── script.js           # 完整的功能实现
└── README.md           # 项目说明
```

## 如何使用

### 本地使用

1. 双击 `index.html` 文件在浏览器中打开
2. 选择需要的同意书类型
3. 阅读同意书内容
4. 双方完成签名
5. 预览并导出为图片或PDF

### 部署到互联网（推荐）

1. 将项目文件上传到GitHub仓库（如 `username.github.io`）
2. 访问 `https://username.github.io` 即可在线使用
3. 任何设备都可以通过这个网址访问

## 功能流程

1. **主页面** → 选择同意书类型
2. **同意书页面** → 阅读内容 → 双方签名
3. **预览页面** → 确认内容 → 导出文件

## 技术实现

- **HTML5**：语义化标签和现代布局
- **CSS3**：现代样式、动画效果和响应式设计
- **JavaScript**：完整的功能实现
- **Canvas API**：手写签名功能
- **html2canvas**：用于截图导出
- **jsPDF**：用于PDF导出

## 核心功能

### 1. 签名功能
- 支持鼠标和触摸输入
- 实时视觉反馈
- 支持清除签名
- 验证双方都已签名

### 2. 导出功能
- 导出为高清图片（PNG格式）
- 导出为PDF文件（支持跨页显示）
- 导出过程中的加载提示
- 导出完成的成功提示

### 3. 响应式设计
- 适配桌面端、平板和移动设备
- 流畅的页面切换动画
- 自适应布局和字体大小
